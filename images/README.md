# Images

This folder stores all screenshots, charts, and visual assets used in the project.

## Naming Convention
```
<type>_<description>_<version>.png
```

Examples:
- `dashboard_executive_summary_v1.png`
- `chart_churn_by_contract.png`
- `chart_churn_reasons_treemap.png`
- `heatmap_state_churn_rate.png`
- `plot_tenure_vs_churn.png`

## Sub-folders (add as needed)
- `dashboard/`  — Power BI dashboard screenshots
- `eda/`        — Python EDA plots
- `model/`      — Model performance plots (confusion matrix, ROC curve)
